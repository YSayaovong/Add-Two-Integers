Add Two Integers - Python Program

Project Overview

This repository contains a simple Python script that adds two integers together and displays the result. The program is designed to demonstrate basic input and arithmetic operations in Python.

How to Run the Program

  1. Clone the repository to your local machine.
  2. Ensure you have Python installed on your system.
  3. Run the script using the following command: 
    python add_two_integers.py
  4. Follow the prompts to enter two integers, and the program will display the sum.
